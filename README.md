# Kindness-Counts: A Cinderella Story

A text-based adventure game built in Python, inspired by the Cinderella fairytale. Players navigate branching storylines through a moral choice system — every decision affects your **Aura Points**, which determine how the story ends.

This is my first Python project.

---

## Overview

The game follows a Cinderella-inspired narrative where the player faces a series of choices, each carrying consequences. Acts of kindness raise your Aura score; selfish decisions lower it. The final ending is determined by your accumulated score across the full playthrough.

---

## Features

- Branching narrative with multiple story paths
- Aura point system that tracks player behaviour across the game
- Four distinct endings based on player choices
- Fully playable in the terminal

---

## Endings

| Ending | Condition |
|---|---|
| Locked in the Attic | Early game-ending choices |
| A Lesson Learned | Low aura score |
| A Night to Remember | Mid-range aura score |
| Happily Ever After | High aura score |

---

## Tech

Python 3 — no external libraries required.
